# Security Policy

## Reporting A Vulnerability

If you believe you have found a security issue on this website, please contact
Kesari by Shammi Randhawa through the contact page:

https://www.kesaribyshammirandhawa.com/contact.html

Please include the affected URL, a clear description of the issue, and safe
steps to reproduce it. Do not exploit, modify, delete, or exfiltrate data.

## Current Scope

This is a static marketing website deployed on Vercel. Security-sensitive
changes should be made through the GitHub repository and deployed through the
approved Vercel pipeline.
