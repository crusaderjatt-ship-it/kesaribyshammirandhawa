# Kesari by Shammi Randhawa

Static clone of the Pomelli-generated Kesari by Shammi Randhawa website,
updated with the latest local edits to `index.html`.

## Run locally

```powershell
npm.cmd run dev
```

Open:

```text
http://127.0.0.1:4173
```

There is no build step right now. The site is served directly from `index.html`
with local image assets in `assets/`.
